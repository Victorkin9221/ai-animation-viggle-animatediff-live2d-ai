## 🔧 Troubleshooting
If the program does not start, check these common items:

**Program fails to open**
Check if your antivirus software is blocking the application. Some security settings block new programs from running without permission. Add an exception for the folder if necessary.

**High usage warnings**
Animation software tasks the graphics card heavily. If the screen becomes unresponsive, close other graphics-heavy programs like web browsers or games while the animation renders.

**Incorrect output**
If the animation looks distorted, check your input image. Ensure the background is clear or simple. High-contrast images with clean edges work best for motion generation.
